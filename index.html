Annotation

GroupDocs.Annotation is a .NET and Java document annotation and collaboration tools set that supports all popular document formats and many different annotation types. Annotate documents (Annotate DOCX, XLSX, PPTX, PDF and other document formats), edit annotations, collaborate with teammates, customize your annotations, add comments and much more!
